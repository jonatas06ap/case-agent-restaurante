# FAQ

A tela de **FAQ** reúne tutoriais em vídeo e respostas rápidas para as principais dúvidas sobre a plataforma Dionísio.

## Como acessar

No menu lateral esquerdo, clique em **FAQ**.

***

## O que você encontra

A tela exibe uma lista de tutoriais organizados por categoria. Cada item mostra o tema, uma breve descrição e a duração do vídeo. Clique em qualquer tutorial para expandir e assistir.

Use a **barra de busca** para localizar um assunto específico, ou os **Filtros** para navegar por categoria.
