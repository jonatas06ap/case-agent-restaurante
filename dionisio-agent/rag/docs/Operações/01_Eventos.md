# Eventos

A tela de **Eventos** é onde você cria e gerencia eventos do seu estabelecimento — festas, shows, noites temáticas e qualquer outra programação especial com lista de participantes e controle de inscrições.

## Como acessar

No menu lateral esquerdo, clique em **Eventos**.

***

## O que você encontra nesta tela

* **Link dos Eventos Públicos** — URL pública para compartilhar com clientes para que visualizem e se inscrevam nos eventos disponíveis. Clique no ícone de copiar para copiar o link.
* **Configuração de Coleta de Dados** — toggle para desativar a coleta de dados pessoais dos participantes quando necessário.
* **Aba Próximos Eventos** — lista de eventos futuros com data, capacidade e status.
* **Aba Eventos Passados** — histórico de eventos realizados.

***

## Como criar um evento

Clique em **+ Cadastrar evento**. O formulário tem 5 passos.

### 1️⃣ Passo 1 — Informações Básicas

| Campo                 | Obrigatório | Descrição                                                                                           |
| --------------------- | ----------- | --------------------------------------------------------------------------------------------------- |
| Nome do Evento        | **Sim**     | Título do evento. Máximo 100 caracteres.                                                            |
| Descrição do Evento   | **Sim**     | Descrição completa. Máximo 500 caracteres.                                                          |
| Data do Evento        | **Sim**     | Data em que o evento ocorrerá.                                                                      |
| Horário de Início     | **Sim**     | Hora de início.                                                                                     |
| Horário de Término    | **Sim**     | Hora de encerramento. Se menor que o início, o sistema entende que o evento vai até o dia seguinte. |
| Fechamento das Listas | Não         | Data e hora limite para inscrições. Se deixado em branco, usa o horário de término do evento.       |

### 2️⃣ Passo 2 — Ingressos e Listas

Configure os tipos de ingresso ou listas de acesso para o evento:

* Defina categorias de ingresso (ex: Pista, VIP, Camarote)
* Configure o valor de cada categoria
* Defina a quantidade disponível por categoria
* Ative ou desative a venda online para cada tipo

### 3️⃣ Passo 3 — Capacidade e Controle

| Campo                   | Descrição                                               |
| ----------------------- | ------------------------------------------------------- |
| Capacidade Total        | Número máximo de participantes no evento.               |
| Controle de Entrada     | Ative para habilitar o check-in por QR Code na entrada. |
| Permitir Fila de Espera | Clientes se inscrevem em espera se o evento lotar.      |

### 4️⃣ Passo 4 — Pagamento

| Campo                  | Descrição                                                            |
| ---------------------- | -------------------------------------------------------------------- |
| Gateway de Pagamento   | Selecione o método de pagamento para ingressos online.               |
| Política de Reembolso  | Defina as condições de cancelamento e estorno informadas ao cliente. |
| Instrução de Pagamento | Texto exibido ao cliente durante a compra. Máximo 300 caracteres.    |

### 5️⃣ Passo 5 — Revisão

Resumo completo da configuração. Revise todos os dados antes de publicar o evento. Clique em **Finalizar** para salvar e publicar.

***

## Gerenciando um evento existente

Clique no evento para acessar:

* **Lista de inscritos** — visualize todos os participantes confirmados
* **Check-in** — registre a entrada dos participantes no dia do evento
* **Estatísticas** — ingressos vendidos, capacidade restante e receita gerada
* **Compartilhar** — copie o link direto do evento para divulgar

> 💡 **Dica:** Ative o Fechamento das Listas com antecedência para dar tempo de organizar a logística de entrada e evitar inscrições de última hora.
