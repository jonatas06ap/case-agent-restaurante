# Links

A tela de **Gerenciar Links** é onde você cria e monitora links rastreáveis para suas campanhas de marketing — acompanhe cliques, analise o desempenho por canal e identifique quais ações geram mais tráfego.

## Como acessar

No menu lateral esquerdo, clique em **Links**.

***

## O que você encontra

No topo da tela há três contadores:

| Métrica             | O que mostra                                             |
| ------------------- | -------------------------------------------------------- |
| **Total de Links**  | Quantidade total de links cadastrados.                   |
| **Links Ativos**    | Links com status ativo disponíveis para uso.             |
| **Links Expirados** | Links que ultrapassaram a data de expiração configurada. |

O campo **Link de Links Rastreáveis** exibe uma URL pública que reúne todos os seus links rastreáveis em uma página acessível pelos clientes.

***

## Como criar um Link

Clique em **+ Novo Link**. Preencha o formulário:

### Informações Básicas

| Campo          | Obrigatório | Descrição                                                                                            |
| -------------- | ----------- | ---------------------------------------------------------------------------------------------------- |
| Título do Link | **Sim**     | Nome descritivo para identificar o link internamente.                                                |
| Código Curto   | **Sim**     | Parte final da URL rastreável (ex: `promo-natal`). Use o botão **Gerar** para criar automaticamente. |
| URL de Destino | **Sim**     | Endereço para onde o cliente será redirecionado ao clicar.                                           |
| Descrição      | Não         | Anotação interna sobre o uso do link.                                                                |

### Configurações

| Campo             | Descrição                                                                   |
| ----------------- | --------------------------------------------------------------------------- |
| Tipo do Link      | Classifica o link por categoria (ex: Promoção, Cardápio, Reservas, Outro).  |
| Status            | Ativo ou Inativo. Links inativos não redirecionam.                          |
| Campanha          | Vincula o link a uma campanha de marketing para agrupamento nos relatórios. |
| Tags              | Rótulos para organização e filtro.                                          |
| Data de Expiração | Data em que o link deixa de funcionar automaticamente. Opcional.            |

### Configurações Avançadas

| Opção                            | O que faz                                            |
| -------------------------------- | ---------------------------------------------------- |
| Rastreamento Habilitado          | Registra cada clique no link.                        |
| Coletar informações do navegador | Registra dados do dispositivo do visitante.          |
| Coletar página de origem         | Registra de qual página o visitante veio (referrer). |
| Coletar localização geográfica   | Registra a cidade/região aproximada do visitante.    |
| Proteger com senha               | Exige senha para acessar o destino do link.          |

Clique em **Criar Link** para salvar.

***

## Monitorando o desempenho

Acesse as estatísticas de um link clicando sobre ele na lista. Você verá o total de cliques, origem dos acessos e evolução ao longo do tempo.

> 💡 **Dica:** Crie um link rastreável diferente para cada canal de divulgação (Instagram, WhatsApp, Google) para descobrir qual canal traz mais clientes.
