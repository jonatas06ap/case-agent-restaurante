# Integrações

A tela de **Integrações** conecta o Dionísio a outras plataformas e ferramentas — sistemas PDV, marketplaces de delivery, ferramentas de marketing e logística. Cada integração é configurada de forma independente.

## Como acessar

No menu lateral esquerdo, clique em **Integrações**.

***

## Abas de Integrações

A tela está organizada em seis categorias:

### PDV

Integre com seu sistema de ponto de venda para sincronizar pedidos e dados operacionais:

| Integração   | Descrição                                              |
| ------------ | ------------------------------------------------------ |
| **Saipos**   | Sincronização de pedidos e dados com o sistema Saipos. |
| **Atena**    | Sincronização com o sistema Atena.                     |
| **Eye PDV**  | Sincronização com o sistema Eye PDV.                   |
| **Colibri**  | Integração com o sistema Colibri.                      |
| **Eclética** | Integração com o sistema Eclética.                     |

### Marketplace

Receba e gerencie pedidos de plataformas externas:

| Integração | Descrição                                                              |
| ---------- | ---------------------------------------------------------------------- |
| **iFood**  | Recebe e gerencia pedidos do iFood automaticamente dentro do Dionísio. |

### Marketing

Conecte ferramentas de rastreamento para otimizar campanhas:

| Integração             | Descrição                                                      |
| ---------------------- | -------------------------------------------------------------- |
| **Facebook Pixel**     | Rastreia conversões e otimiza anúncios no Meta Ads.            |
| **Google Tag Manager** | Gerencia códigos de tracking sem modificar o código do site.   |
| **Google Analytics 4** | Análise avançada de comportamento e insights sobre visitantes. |

### NPS

Colete pesquisas de satisfação via plataformas especializadas:

| Integração | Descrição                                                |
| ---------- | -------------------------------------------------------- |
| **Falae**  | Integração para coleta de pesquisas NPS com os clientes. |

### Logística

Conecte serviços de entrega para gerenciar motoristas e rastreamento:

| Integração                  | Descrição                                                   |
| --------------------------- | ----------------------------------------------------------- |
| **Open Delivery Logística** | Gerenciamento de entregas e rastreamento via Open Delivery. |
| **Lalamove**                | Contratação e rastreamento de entregas via Lalamove.        |

### Importação de Cardápios

Ferramentas para migrar ou importar catálogos de outros sistemas:

| Integração             | Descrição                                                                   |
| ---------------------- | --------------------------------------------------------------------------- |
| **Migração Firestore** | Migre o cardápio da estrutura antiga (Firestore) para a nova estrutura SQL. |
| **Saipos**             | Importe o cardápio diretamente do sistema Saipos.                           |

***

## Como ativar uma integração

Clique no card da integração desejada e siga o passo a passo de autenticação. Cada integração tem seu próprio fluxo — geralmente envolve inserir credenciais ou autorizar acesso à plataforma parceira.

Após a configuração, a integração aparece como **Conectada**.

> 💡 **Dica:** Verifique periodicamente se as integrações estão ativas — algumas podem desconectar se as credenciais expirarem ou se a senha da conta parceira for alterada.
